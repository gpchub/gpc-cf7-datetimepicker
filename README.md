# gpc-cf7-datetimepicker
Add Datetimepicker for contact form 7

## datepicker
`[gpc_datepicker your-date class:your-class placeholder "Choose a date"]`

Required field:

`[gpc_datepicker* your-date class:your-class placeholder "Choose a date *"]`

## datetimepicker
`[gpc_datetimepicker your-date class:your-class placeholder "Choose a date"]`

Required field:

`[gpc_datetimepicker* your-date class:your-class placeholder "Choose a date *"]`
